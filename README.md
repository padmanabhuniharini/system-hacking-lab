System Hacking Lab

Project Overview
This repository contains practical exercises performed as part of the System Hacking laboratory. The experiments were conducted in a controlled virtual environment using Kali Linux and Windows 7 virtual machines. The objective was to gain hands-on experience with various system security concepts, attack techniques, password auditing methods, steganography, and privacy protection tools.
The lab activities focused on understanding how attackers and security professionals interact with operating systems, networks, authentication mechanisms, and user data. All experiments were performed for educational purposes in an isolated environment.

Tools and Technologies

* Kali Linux
* Windows 7 Virtual Machine
* VMware Workstation
* Responder
* Metasploit Framework
* msfvenom
* L0phtCrack
* Steghide
* Privacy Eraser


Work Performed
1. Network Poisoning and Credential Capture
A host-only network was configured between Kali Linux and Windows 7 virtual machines. Using the Responder tool, authentication requests generated by the Windows machine were intercepted. The experiment demonstrated how LLMNR, SMB, and WPAD poisoning can be used to capture NTLMv2 authentication hashes in a controlled environment.

2. Reverse TCP Payload Demonstration
A reverse TCP payload was generated and deployed within the virtual lab environment. A listener was configured on the Kali Linux machine, and a successful connection was established between the target and listener systems. This exercise demonstrated the concept of remote session establishment in a controlled setting.

3. Password Auditing
L0phtCrack was used to perform password auditing on local Windows user accounts. Password hashes were analyzed using dictionary and brute-force auditing techniques to evaluate password strength and identify weak credentials.

4. Steganography
Steghide was used to hide confidential information inside an image file. The hidden data was later extracted using the correct passphrase, demonstrating the practical implementation of steganography techniques for data concealment and recovery.

5. Privacy Trace Analysis and Removal
Privacy Eraser was installed on the Windows virtual machine to analyze privacy-related traces. The tool identified browser history, cookies, temporary files, Windows logs, and registry traces. The detected artifacts were securely removed to improve privacy and reduce recoverable system traces.



Learning Outcomes

Through this laboratory work, the following concepts were explored:

* Network authentication mechanisms
* Name resolution poisoning attacks
* Authentication hash interception
* Password auditing and assessment
* Reverse TCP communication concepts
* Steganography and data hiding
* Privacy trace identification and removal
* Security tool usage in virtualized environments
